# 安装说明（Chrome / Microsoft Edge / Mozilla Firefox）

整套扩展以未打包目录交付，三家浏览器都用「加载本地未打包扩展」的方式装载，无需翻墙、无需商店上架、无需签名。

## 目录路径

- macOS：`/Users/你的用户名/Downloads/gpt-js/ChatGPT全能助手-扩展版/`
- Windows：`D:\Downloads\gpt-js\ChatGPT全能助手-扩展版\`（按你的下载位置替换）

下文以「扩展目录」代指该目录。

## Chrome 安装步骤

### macOS

1. 打开 Chrome，地址栏输入 `chrome://extensions/` 回车。
2. 右上角打开「开发者模式」开关。
3. 左上角点「加载已解压的扩展程序」。
4. 在弹出选择框中定位到「扩展目录」，点「选择」。
5. 扩展列表里出现「ChatGPT 全能助手 · Specimen」，工具栏旁出现橙红 K 图标即装载成功。
6. 打开 https://chatgpt.com 验证：右下角应出现 Specimen 浮窗按钮。

### Windows

步骤同 macOS，仅路径写法改为 `D:\...` 即可。Windows 上 Chrome 的开发者模式按钮位置一致。

## Microsoft Edge 安装步骤

Edge 与 Chrome 同为 Chromium 内核，流程几乎一致。

1. 地址栏输入 `edge://extensions/` 回车。
2. 左下角打开「开发人员模式」开关。
3. 点击「加载解压缩的扩展」按钮，选择「扩展目录」。
4. 装载完成后地址栏右侧会出现橙红 K 图标。

macOS / Windows 步骤完全相同。

## Firefox 安装步骤（临时加载）

Firefox 正式版要求扩展走 Mozilla 签名才能永久安装，未打包目录只能临时加载（关掉浏览器后失效，需要再次加载）。如需长期使用，推荐用 Firefox Developer Edition 或 Nightly，可在 `about:config` 关闭签名校验。

### macOS / Windows 通用步骤

1. 打开 Firefox，地址栏输入 `about:debugging#/runtime/this-firefox` 回车。
2. 点击「临时载入附加组件…」按钮。
3. 在弹出选择框中**进入「扩展目录」并选中 `manifest.json` 文件**（注意是文件不是目录，与 Chrome 不同），点「打开」。
4. 「临时扩展」列表里出现「ChatGPT 全能助手 · Specimen」即装载成功。
5. 打开 https://chatgpt.com 验证浮窗按钮。

### 长期安装（可选）

如果你已有 Mozilla AMO 开发者账号，可将整个扩展目录打包成 ZIP，再用 `web-ext sign --api-key=... --api-secret=...` 走 Mozilla 签名流程生成正式 `.xpi`。本目录未自带 `web-ext.config.js`，需自行 `npm i -g web-ext` 后在扩展目录内执行 `web-ext sign`。

## 触发方式

装载后任选其一即可打开助手面板：

- 右下角悬浮按钮：打开 ChatGPT 后右下角的橙红圆形按钮，可拖拽到任意位置。
- 浏览器工具栏图标：点击地址栏右边的橙红 K 图标。
- 页面右键菜单：在 ChatGPT 页面任意位置右键，选「打开 CKNB ChatGPT 全能助手」。
- 快捷键：`Cmd + Shift + K`（macOS）/ `Ctrl + Shift + K`（Windows / Linux）。

## 新功能（v2.3.0）·「导入 · 转换」Tab

进入面板后切到第 4 个 Tab「04 导入 · 转换」，三种输入方式任选其一：

1. **粘贴 JSON** —— 直接把别人给的 JSON 文件内容贴进大文本框。
2. **上传文件** —— 点「选择文件」选本地 `.json` 文件，自动读入并解析。
3. **读剪贴板** —— 点按钮一键拉取剪贴板内容（首次需授权剪贴板读取权限）。

支持自动识别 11 种来源格式（Sub2API / Cockpit / auth.json / CPA / 9router / AxonHub / Codex-Manager / 原 Session / Codex Auth / 裸 JWT / 数组形式），识别失败时可从「来源格式」下拉手动覆盖。

解析后可：
- 多账号场景下点账号 chip 切换预览
- 在 9 种目标格式中选择任意一种「复制」/「下载」
- 「打包此账号 9 种」一次性导出当前账号所有格式
- 「批量 · 全部账号」按当前选中格式，把所有账号一次性导出

## 卸载

- Chrome / Edge：在 `chrome://extensions/` 或 `edge://extensions/` 找到本扩展，点「移除」。
- Firefox：关闭浏览器即可（临时加载机制）；或在 `about:debugging` 中手动点「移除」。

## 常见问题

### 浮窗没出现

1. 确认地址确实是 `https://chatgpt.com/*` 或 `https://chat.openai.com/*`，其他域名不会注入。
2. 打开页面后等待 1～2 秒（脚本走 `document_idle`，需要等待页面渲染完毕）。
3. 检查浏览器控制台是否有报错；若控制台无 `cknb-specimen` 字样，说明 content script 未加载，重新装载扩展。

### 工具栏图标点击没反应

1. 确认当前标签页在 ChatGPT 站点。非匹配站点点击会自动跳转到 ChatGPT 主页。
2. 检查扩展管理页中本扩展是否「已启用」。

### 复制按钮无效

油猴版用 `GM_setClipboard`，扩展环境会降级到 `navigator.clipboard`，需要在 HTTPS 安全上下文执行。ChatGPT 全站 HTTPS，理论上不会失败；如失败请检查浏览器是否拦截了剪贴板权限弹窗。

### 与油猴版冲突？

如果同一浏览器既装了 Tampermonkey 油猴脚本，又装了本扩展，脚本顶部的 `window.__CKNB_TOOLBOX_LOADED__` 守卫会让第二个加载者直接返回，不会重复注入浮窗，但建议二选一以避免菜单/快捷键重复触发。
